Explicacion de CSS

Image Sprite es una tecnica que consiste en usar varias imagenes (iconos, logos, imagenes, etc...) inmersas en una sola imagen para mejorar el rendimiento del servidor y no tener que hacer varias consultas a varias imagenes.
La imagen es diseñada de manera que sea facil definir las posiciones de las imagenes inmersas y ocupar solo es espacio requerido de la imagen para lograr usar la misma. Esta página es ejemplo de ello.
